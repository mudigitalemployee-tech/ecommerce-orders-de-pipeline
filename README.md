## Pipeline Stages

| Stage | Script | Description |
|-------|--------|-------------|
| Extract | `airflow_dag.py` | Reads from source CSV/DB/API |
| Transform | `transform.py` | Dedup, null fill, type cast, PII masking |
| DQ Gate | `dq_checks.py` | Validates quality before Silver write |
| Load | `airflow_dag.py` | Writes Silver + Gold layers |
| Orchestrate | `airflow_dag.py` | Airflow DAG with retries + alerting |
